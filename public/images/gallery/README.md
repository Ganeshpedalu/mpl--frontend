# Gallery Images

Place your gallery images in this folder with the following names:

1. `opening-ceremony.jpg` - Opening Ceremony image
2. `match-action.jpg` - Match Action image
3. `team-celebration.jpg` - Team Celebration image
4. `award-night.jpg` - Award Night image
5. `finals-match.jpg` - Finals Match image
6. `trophy-moment.jpg` - Trophy Moment image

**Supported formats:** JPG, JPEG, PNG, WebP

**Recommended size:** 
- Main slider: 1920x1080px (16:9 aspect ratio)
- Thumbnails: 400x400px (1:1 aspect ratio)

The gallery will automatically display these images. If an image is missing, it will show a gradient fallback with the image title.
